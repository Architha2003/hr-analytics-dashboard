# 📊 HR Analytics Dashboard: Attrition & Workforce Insights

## 🎯 Project Goal
Built an interactive Power BI dashboard to help HR leaders identify attrition risks and allocate retention budgets effectively.

## 🔧 Tools Used
- **Google Sheets**: Data cleaning, IF formulas, Pivot Tables
- **Power BI Desktop**: DAX measures, interactive visuals, dashboard design
- **GitHub**: Version control & portfolio documentation

## 📁 Data Source
[IBM HR Analytics Attrition Dataset](https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset)
- 1,470 employee records
- 35 features: demographics, job roles, compensation, attrition status

## 📈 Key Insights
| Metric | Finding | Business Impact |
|--------|---------|----------------|
| **Attrition Rate** | 17.4% overall | R&D & HR: ~19% attrition \| Sales: 12.3% |
| **Tenure Risk** | 0-2 years: 21% attrition | 2x higher than 5+ years (9%) |
| **Recommendation** | Focus onboarding in R&D/HR | Potential 15% early-attrition reduction |

## 🖥️ Dashboard Features
✅ Executive KPI cards: Total Employees, Attrition Rate %, Avg Tenure  
✅ Interactive filters: Department, Tenure Group, Attrition Status  
✅ Color-coded visuals: Red = high risk, Green = stable

##  How to Use
1. Download `03_Dashboard_Files/HR_Analytics_Dashboard_FINAL.pbix`
2. Open in Power BI Desktop (free download)
3. Use left-panel slicers to filter by department, tenure, or attrition status
4. Hover over charts for detailed tooltips

## 💡 Key Business Recommendations
1. **Focus retention programs on R&D and HR departments** (19% attrition vs 12% in Sales)
2. **Intervene with new hires within first 90 days** (0-2 year tenure group has 21% attrition)
3. **Document Sales team's retention practices** and replicate in high-risk departments
